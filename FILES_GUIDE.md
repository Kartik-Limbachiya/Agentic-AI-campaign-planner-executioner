- **Key classes:** CampaignOrchestrator
- **Key methods:**
  - `plan_campaign()` - AI campaign planning
  - `schedule_campaigns()` - Calendar scheduling
  - `execute_scheduled_campaigns()` - Run campaigns
  - `track_performance()` - Analytics
  - `generate_performance_report()` - Reporting
  - `run_complete_workflow()` - Full end-to-end

### campaign_planner.py
- **Role:** AI-powered campaign planning
- **Key classes:** CampaignPlanner
- **Features:**
  - GPT-4 integration (optional)
  - Platform-specific strategies
  - Content generation per platform
  - Fallback mode without API

### calendar_manager.py
- **Role:** Calendar and scheduling
- **Key classes:** CampaignCalendar, CampaignEvent, Platform
- **Features:**
  - Multi-day scheduling
  - Platform optimization
  - Calendar export (JSON)
  - Event tracking

### campaign_executor.py
- **Role:** Campaign execution simulation
- **Key classes:** CampaignExecutor, SocialMediaSimulator
- **Features:**

  - Realistic metrics
  - Execution tracking
  - Performance data generation

### performance_tracker.py
- **Role:** Analytics and reporting
- **Key classes:** PerformanceAnalyzer
- **Features:**
  - Metrics tracking
  - Report generation

  - Analytics export

### __init__.py
- **Role:** Package initialization
